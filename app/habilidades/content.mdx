### Tailwind CSS

Já utilizei muitos frameworks em busca de uma melhor produtividade, como: Bootstrap, Chackra UI, Next UI, Material UI, mas foi com o Tailwind que mais rápido me adaptei.

### HeadlessCMS

Quando necessário um sistema CMS para administrar o conteúdo dos sites, utilizo ferramentas de Headless CMS, que agilizam muito o progresso dos projetos, pois não preciso dedicar muito tempo em contrução de sistemas do zero.

### Next.js

Next.js é o framework React que escolhi para me especializar. Tenho muita confiança de que posso entregar um ótimo produto com esse framework.
